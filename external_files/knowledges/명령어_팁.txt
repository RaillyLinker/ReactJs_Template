[ReactJs 명령어]
- 리엑트 프로젝트 생성 (TypeScript 설정)
    >> npx create-react-app ${프로젝트 폴더 이름} --template typescript

- 리엑트 프로젝트 node_modules 생성 (package.json dependencies 수정 후 실행하면 라이브러리가 설치됩니다.)
    >> npm install

- 리엑트 프로젝트 실행 (리엑트 프로젝트 폴더 경로로 이동 후)
    >> npm start


[VsCode 단축키]
